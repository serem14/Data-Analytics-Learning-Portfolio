Victor Kiptoo – Data Analysis Portfolio Project

This folder contains my data analysis project involving two areas:

1. Medical Risk Analysis
2. Customer Behaviour Analysis

The work includes data cleaning, data transformation, classification, and dashboard visualization using Excel and Power BI.

Files in this folder:

* Victor_Kiptoo_Analytics_Dashboard.pbix
  Contains two dashboard pages:
  • Medical Risk Dashboard
  • Customer Behaviour Dashboard

* Cleaned_Datasets folder
  Contains the cleaned Excel datasets used for analysis.

* Screenshots folder
  Contains images of the dashboards for quick preview.

* Project_Description
  Explains the objective, process, and key findings of the project.

How to view:
Open the PBIX file using Power BI Desktop to interact with the dashboards. Screenshots are provided for quick reference if Power BI is not available.

This project demonstrates practical skills in:
Data cleaning, analysis, classification, visualization, and interpretation.
